## Requirements

- Xcode 15 or later
- iOS 17.0 or later
- Beeminder account with API access

## Setup

1. Clone this repository
2. Open `beeminder_widget.xcodeproj` in Xcode
3. Update your Beeminder credentials in `BeeminderAPI.swift`
4. Configure App Groups in Xcode for both targets
5. Build and run!
